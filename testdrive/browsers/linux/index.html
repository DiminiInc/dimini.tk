Методика сравнения

Результаты браузеров в категориях: основное, функциональность, производительность рассчитываются как среднее значение по каждому из указанных параметров. Итоговый результат рассчитывается как среднее значение результатов по этим категориям, взятых с коэффициентами соответствующими важности данной категории.

Основное

  • Статус разработки
  • Частота обновлений
  • Популярность
  • Размер установщика
  • Время запуска
  • Потребление времени процессора
  • Потребление оперативной памяти
  • Потребление места на диске
  • Размер рабочей области
  • Открытость исходного кода
  • Поддержка операционных систем
  • Поддержка языков

Функциональность

  • Стартовая страница
  • Работа с вкладками
  • История
  • Закладки
  • Расширения
  • Инструменты разработчика
  • Синхронизация
  • Стабильность
  • Поддержка веб-технологий и протоколов
